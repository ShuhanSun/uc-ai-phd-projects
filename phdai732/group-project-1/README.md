# Group Project 1 - Credit Risk Analytics System

This project focuses on developing an AI-driven credit card default prediction system using machine learning and data analytics techniques.

## Objectives

- Explore and preprocess financial data
- Perform exploratory data analysis (EDA)
- Build and evaluate classification models
- Analyze ethical considerations in AI-driven financial systems
- Collaborate using GitHub and Google Colab

## Main Notebook

- `credit_default_analysis.ipynb`

## Technologies

- Python
- pandas
- scikit-learn
- matplotlib
- seaborn
- Google Colab

## Team Workflow

The project is collaboratively developed using GitHub for version control and Google Colab for notebook-based experimentation and analysis.
